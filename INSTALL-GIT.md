# Install Git to Deploy Your Website

## Quick Installation

1. **Download Git for Windows:**
   - Visit: https://git-scm.com/download/win
   - Click "Download" button

2. **Install Git:**
   - Run the downloaded installer
   - Use all default settings (just click "Next" through each step)
   - Click "Install" when ready

3. **Restart Cursor:**
   - Close and reopen Cursor after installation
   - Or restart your terminal/PowerShell

4. **Run the deployment script:**
   - In PowerShell, run: `.\deploy-to-github.ps1`
   - Or tell the AI assistant "run automatically" again
